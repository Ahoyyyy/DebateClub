DebateClub
==========

A Unity3D based 2D fighting game

How to load
===========
Create a new project within unity and import all the files into the project directory.

Requirements
===========
Set up MonoDevelop to not use Unix-style line endings (\n) instead of the windows default (\n\r) by setting the option in 

**Tools > Options > Source Code > Code Formatting > Text File > Line Endings = Unix**
